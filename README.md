# Event Manager (Next.js 14 + TypeScript)

Тестовое приложение для управления мероприятиями: просмотр, создание, редактирование, удаление, фильтрация и сортировка.

## Стек

- Next.js 14 (App Router)
- React 18
- TypeScript
- CSS (без UI-библиотек)
- localStorage для хранения данных

## Реализованные возможности

### Основные
- Список мероприятий в виде карточек.
- Добавление мероприятия с валидацией:
  - название обязательно;
  - дата и время обязательны;
  - дата не может быть в прошлом.
- Редактирование существующего мероприятия.
- Удаление мероприятия с подтверждением (`window.confirm`).
- Фильтрация по категории и статусу.
- Сортировка по дате или названию (asc/desc).

### Дополнительные
- Поиск по названию и описанию.
- Избранное (переключатель звезды, отдельный фильтр `Favorites only`).
- Экспорт списка мероприятий в JSON.

## Архитектура

- `app/page.tsx` — Server Component, рендерит страницу и передаёт начальные данные.
- `components/event-manager.tsx` — Client Component с интерактивной логикой:
  - CRUD,
  - валидация формы,
  - фильтры/сортировка,
  - синхронизация c `localStorage`.
- `lib/types.ts` — строгие типы и перечисления.
- `lib/default-events.ts` — мок-данные для первичного рендера.

## Запуск

```bash
npm install
npm run dev
```

Открыть: `http://localhost:3000`.

## Продакшен-сборка
